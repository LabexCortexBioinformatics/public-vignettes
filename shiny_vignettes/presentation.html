LabEx CORTEX Bioinformatics platform

Activity

The LabEx CORTEX Bioinformatics platform composed of 2 bioinformaticians who offer an expertise in single cell data analysis.

Aims

  • Analysis of data obtained through single cell approach
  • Analysis of data obtained through single nuclei approach

Networking

The bioinformatics platform has conducted collaborations with all the teams of the CORTEX project doing single cell data analysis.

Services

  • Helping teams of biologists with single cell data analysis
  • Providing a visual web interface for plot visualisation of filtered and processed single cell datasets

Equipment

The platform disposes of a workstations containing 40 computationnal cores, 16To storage system, and a portable computer for analysis when going in other labs.
